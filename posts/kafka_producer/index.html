Kafka Producer

л©”мӢңм§Җ л°ңн–ү/кө¬лҸ… мӢңмҠӨн…ңм—җм„ң ProducerлҠ” л©”мӢңм§Җмқҳ л°ңн–үмқ„ мҲҳн–үн•ҳлҠ” м»ҙнҸ¬л„ҢнҠёмқҙлӢӨ.

мқҙ кёҖм—җм„ңлҠ” Kafka Client APIлҘј мӮ¬мҡ©н•ҳм—¬ Kafka ProducerлҘј л§Ңл“Өм–ҙліҙкі , л©”мӢңм§Җк°Җ мӢӨм ңлЎң м–ҙл–Ө кіјм •мқ„ кұ°міҗ м „лӢ¬лҗҳлҠ”м§Җ м•Ңм•„ліёлӢӨ.

build.gradleм—җ Kafka Client мқҳмЎҙм„ұ м¶”к°Җн•ҳкё°

build.gradleмқҳ dependencies лё”лЎқм—җ м•„лһҳ кө¬л¬ёмқ„ м¶”к°Җн•ҳм—¬ Kafka Client API мқҳмЎҙм„ұмқ„ м¶”к°Җн•ңлӢӨ. мҳӨлҠҳ лӮ м§ң(2020л…„ 5мӣ” 2мқј) кё°мӨҖмңјлЎң 2.5.0 лІ„м „(Release: 2020/04)мқҙ к°ҖмһҘ мөңмӢ мқҳ APIмқҙл©°, кіөмӢқ л¬ём„ңм—җм„ңлҸ„ мӮ¬мҡ©мқ„ к¶ҢмһҘн•ңлӢӨ.

1

compile group: 'org.apache.kafka', name: 'kafka-clients', version: '2.5.0'

KafkaProducer к°қмІҙ мҙҲкё°нҷ”

KafkaProducerлҠ” Kafka Clusterм—җ л©”мӢңм§ҖлҘј л°ңн–үн•ҳлҠ” APIлҘј м ңкіөн•ҳлҠ” нҒҙлһҳмҠӨмқҙлӢӨ.

KafkaProducer к°қмІҙлҘј мғқм„ұн•  л•ҢлҠ” Kafka Cluster м—°кІ° м •ліҙ, Serializer м„Өм • л“ұмқ„ м „лӢ¬н•ҙмЈјм–ҙм•ј н•ңлӢӨ. мқҙлҹ¬н•ң м •ліҙл“ӨмқҖ java.util.PropertiesлҘј нҶөн•ҙ м „лӢ¬н•ңлӢӨ.

  • bootstrap.servers: Kafka Clusterмқҳ Broker мЈјмҶҢлҘј (host:port) нҳ•мӢқмңјлЎң л„Јм–ҙмЈјл©ҙ лҗңлӢӨ. Kafka ClusterлҘј кө¬м„ұн•ҳлҠ” Broker мӨ‘ н•ҳлӮҳмқҳ мЈјмҶҢл§Ң л„ҳкІЁмӨҳлҸ„ м—°кІ° кіјм •м—җм„ң лӢӨлҘё Brokerл“Өмқҳ мЈјмҶҢлҸ„ мҲҳмӢ н•ңлӢӨ. лӢӨл§Ң мҡҙмҳҒ нҷҳкІҪм—җм„ңлҠ” Producer мһ¬мӢңмһ‘ мӢң кё°мһ…лҗң мЈјмҶҢмқҳ Brokerк°Җ лӢӨмҡҙлҗҳлҚ”лқјлҸ„ лӢӨлҘё Broker мЈјмҶҢлҘј нҶөн•ҙ м—°кІ° к°ҖлҠҘн•ҳлҸ„лЎқ 2к°ң мқҙмғҒмқҳ мЈјмҶҢлҘј л„Јм–ҙмЈјлҠ” кІғмқ„ к¶ҢмһҘн•ңлӢӨ.
  • key.serializer: л©”мӢңм§Җмқҳ KeyлҘј м§Ғл ¬нҷ” н•  мҲҳ мһҲлҠ” Serializer нҒҙлһҳмҠӨлҘј м„Өм •н•ңлӢӨ.
  • value.serializer: л©”мӢңм§Җмқҳ ValueлҘј м§Ғл ¬нҷ” н•  мҲҳ мһҲлҠ” Serializer нҒҙлһҳмҠӨлҘј м„Өм •н•ңлӢӨ.

Properties к°қмІҙ мҙҲкё°нҷ”к°Җ мҷ„лЈҢлҗҳм—ҲлӢӨл©ҙ, мқҙм ң KafkaProducer к°қмІҙлҘј м•„лһҳмҷҖ к°ҷмқҙ мҙҲкё°нҷ”н•ңлӢӨ. Producerмқҳ Generic к°’мқҖ л©”мӢңм§ҖлҘј кө¬м„ұн•ҳлҠ” Key, Value нғҖмһ…мқ„ лӮҳнғҖлӮёлӢӨ.

Producer<String, String> kafkaProducer = new KafkaProducer<>(properties);

м°ёкі лЎң Producer(KafkaProducer)лҠ” Thread-Safeн•ң к°қмІҙмқҙлӢӨ. л”°лқјм„ң н•ҙлӢ№ к°қмІҙлҘј SingletonмңјлЎң мң м§Җн•ҳм—¬ м• н”ҢлҰ¬мјҖмқҙм…ҳ лӮҙмқҳ Threadл“Өмқҙ кіөмң н•ҳлҸ„лЎқ н•ҳлҠ” кІғмқҙ мўӢлӢӨ. мҳӨнһҲл Ө л§Һмқҙ л§Ңл“ӨмҲҳлЎқ Kafka Broker ServerмҷҖмқҳ Connectionл§Ң л§Һмқҙ мғқм„ұлҗҳкё° л•Ңл¬ём—җ мўӢм§Җ м•ҠмқҖ мҳҒн–Ҙмқ„ лҒјм№ҳкІҢ лҗңлӢӨ.

send() л©”м„ңл“ңлЎң л©”мӢңм§Җ ліҙлӮҙкё°

KafkaProducerмқҳ мҙҲкё°нҷ”к°Җ мҷ„лЈҢлҗҳм—ҲлӢӨл©ҙ send() л©”м„ңл“ңлҘј нҶөн•ҙ Kafka BrokerлЎң л©”мӢңм§ҖлҘј м „мҶЎн•  мҲҳ мһҲлӢӨ.

send л©”м„ңл“ңлЎң м „лӢ¬н•ҳлҠ” к°қмІҙлҠ” ProducerRecordмқҙлӢӨ. кё°ліём ҒмңјлЎң TopicлӘ…кіј л©”мӢңм§Җмқҳ Value л§ҢмңјлЎңлҸ„ ProducerRecord к°қмІҙлҘј л§Ңл“Өм–ҙ м „мҶЎмқҙ к°ҖлҠҘн•ҳл©°, м¶”к°Җм ҒмңјлЎң Key, Partition ID, Timestamp, Header л“ұмқ„ м„Өм •н•  мҲҳ мһҲлӢӨ.

мҙқ 100к°ңмқҳ л©”мӢңм§Җк°Җ м •мғҒм ҒмңјлЎң л“Өм–ҙмҳЁ кІғмқ„ нҷ•мқён•  мҲҳ мһҲлӢӨ.

sendлҠ” л№„лҸҷкё° л©”м„ңл“ңмқҙлӢӨ.

send л©”м„ңл“ңлҠ” л№„лҸҷкё° л©”м„ңл“ңмқҙлӢӨ. Kafka Client Javadoc м—җм„ңмқҳ send л©”м„ңл“ң мЈјм„қмқ„ ліҙл©ҙ м•„лһҳмҷҖ к°ҷмқҙ м„ӨлӘ…лҗҳм–ҙ мһҲлӢӨ.

Asynchronously send a record to a topic and invoke the provided callback when the send has been acknowledged.

The send is asynchronous and this method will return immediately once the record has been stored in the buffer of records waiting to be sent.

send л©”м„ңл“ңлҠ” л№„лҸҷкё°лЎң лҸҷмһ‘н•ҳл©°, м „лӢ¬лҗң ProducerRecord л©”мӢңм§ҖлҠ” BrokerлЎң мҰүмӢң м „мҶЎлҗҳлҠ” кІғмқҙ м•„лӢҲлқј Producer н”„лЎңм„ёмҠӨмқҳ bufferм—җ м ҖмһҘлҗң нӣ„ м „мҶЎлҗңлӢӨ. мқҙмҷҖ к°ҷмқҖ кө¬нҳ„мқ„ н•ң мқҙмң лҠ” л°”лЎң лӢӨмқҢ мӨ„м—җ м ҒнҳҖмһҲлӢӨ.

This allows sending many records in parallel without blocking to wait for the response after each one.

лҸҷкё° л°©мӢқмңјлЎң л©”м„ңл“ңлҘј кө¬нҳ„н–ҲлӢӨл©ҙ sendлҘј нҳём¶ңн•  л•Ңл§ҲлӢӨ н•ҳлӮҳм”© л©”мӢңм§Җк°Җ BrokerлЎң м „мҶЎлҗ  кІғмқҙлӢӨ. мқҙлҠ”

  • send л©”м„ңл“ң нҳём¶ң мӢң I/O Blockмқҙ л°ңмғқн•ҳм—¬ лі‘л ¬м„ұмқҙ л–Ём–ҙм§җ
  • BulkлЎң ліҙлӮҙлҠ” кІғліҙлӢӨ Network Overheadк°Җ л°ңмғқ

н•ҳлҠ” лӢЁм җмқҙ мЎҙмһ¬н•ңлӢӨ.

л•Ңл¬ём—җ KafkaProducerмқҳ sendлҘј нҳём¶ңн•ҳкІҢ лҗҳл©ҙ ProducerRecordлҘј KafkaProducerмқҳ лӮҙл¶Җ Bufferм—җ м ҖмһҘн•ҙлҶ“мқҖ нӣ„, м–ҙлҠҗм •лҸ„ л©”мӢңм§Җк°Җ лӘЁмқҙкІҢ лҗҳл©ҙ м—¬лҹ¬ к°ңмқҳ л©”мӢңм§ҖлҘј н•ңлІҲм—җ ліҙлӮҙкі  Callbackмқ„ нҳём¶ңн•ңлӢӨ.

sendмқҳ Callback нҷңмҡ©н•ҳкё°

sendмқҳ л©”м„ңл“ң мӢңк·ёлӢҲмІҳлҠ” м•„лһҳмҷҖ к°ҷлӢӨ.

Future<RecordMetadata>к°Җ л°ҳнҷҳлҗҳкё° л•Ңл¬ём—җ getмқ„ нҳём¶ңн•ҳл©ҙ н•ҙлӢ№ Callмқҙ Blockлҗҳм–ҙ лҸҷкё° л°©мӢқмңјлЎң л©”мӢңм§Җ м „мҶЎкІ°кіјлҘј м•Ң мҲҳ мһҲкІ м§Җл§Ң, мқҙлҠ” Kafka Client кө¬нҳ„мһҗмқҳ мқҳлҸ„к°Җ м•„лӢҗ кІғмқҙлӢӨ. мқҙмғҒм Ғмқё л©”мӢңм§Җ м „мҶЎ нӣ„мІҳлҰ¬ л°©мӢқмқҖ Callbackмқ„ мқҙмҡ©н•ҳлҠ” кІғмқҙлӢӨ.

CountDownLatchлҘј нҶөн•ҙ лӘЁл“  л©”мӢңм§Җк°Җ мІҳлҰ¬лҗң мқҙнӣ„м—җ м• н”ҢлҰ¬мјҖмқҙм…ҳмқ„ мў…лЈҢн•  мҲҳ мһҲлҸ„лЎқ н•ҳмҳҖлӢӨ. н”„лЎңлҚ•м…ҳ нҷҳкІҪм—җм„ңлҠ” мң„мҷҖ к°ҷмқҙ мқөлӘ… мқён„°нҺҳмқҙмҠӨлЎң кө¬нҳ„н•ҳлҠ” кІғліҙлӢӨлҠ” мӢӨнҢЁ мӢң мһ¬мӢңлҸ„ к°ҖлҠҘн•ң лЎңм§Ғмқ„ кө¬нҳ„н•ң кө¬мІҙ нҒҙлһҳмҠӨлЎң мІҳлҰ¬н•ҳлҠ” кІғмқҙ лҚ” мўӢмқ„ кІғ к°ҷлӢӨ.

sendлҠ” мӢӨм ңлЎң м–ҙл–»кІҢ лҸҷмһ‘н•ҳлҠ”к°Җ?

sendлҘј нҶөн•ң л©”мӢңм§Җ м „мҶЎм—җ кҙҖм—¬н•ҳлҠ” к°қмІҙлҠ” SenderмҷҖ RecordAccumulatorмқҙлӢӨ. sendлҘј нҳём¶ңн•ҳкІҢ лҗҳл©ҙ к°„лһөнһҲ м•„лһҳмҷҖ к°ҷмқҖ мҲңм„ңлЎң л©”мӢңм§Җ м „мҶЎмқҙ мҲҳн–үлҗңлӢӨ.

Kafka Clusterм—җм„ң Topic кҙҖл Ё мөңмӢ  л©”нғҖлҚ°мқҙн„°лҘј Fetch н•ңлӢӨ.

KafkaProducerлҠ” мһҗмӢ мқҙ лҚ°мқҙн„°лҘј м „мҶЎн•  Topicкіј нҢҢнӢ°м…ҳ л“ұмқҳ м •ліҙлҘј кҙҖлҰ¬н•ҳкё° мң„н•ҙ ProducerMetadataлҘј мң м§Җн•ңлӢӨ. л©”мӢңм§ҖлҘј м „мҶЎн•ҳкё° м „м—җ waitOnMetadata л©”м„ңл“ңлҘј нҳём¶ңн•ҳм—¬ мөңмӢ  л©”нғҖлҚ°мқҙн„°лҘј Fetchн•ң л’Ө ProducerMetadata мғҒнғңлҘј к°ұмӢ н•ңлӢӨ.

мң„мқҳ лҚ°мқҙн„°лҠ” waitOnMetadata нҳём¶ң мқҙм „мқҳ ProducerMetadataмқҙл©°, м•„лһҳмқҳ лҚ°мқҙн„°лҠ” waitOnMetadata нҳём¶ң мқҙнӣ„мқҳ ProducerMetadataмқҙлӢӨ.

л©”мӢңм§ҖлҘј м „мҶЎн•  Topicмқё testмқҳ м •ліҙк°Җ мғқкІЁлӮң кІғмқ„ нҷ•мқён•  мҲҳ мһҲлӢӨ. м°ёкі лЎң waitOnMetadataлҠ” мҡ”мІӯн•ң Topicм—җ лҢҖн•ҙм„ңл§Ң м •ліҙлҘј м—…лҚ°мқҙнҠён•ңлӢӨ. лҳҗн•ң waitOnMetadata кіјм •м—җм„ң Cluster к°қмІҙ лҳҗн•ң м—…лҚ°мқҙнҠён•ңлӢӨ. мқҙ м •ліҙлҸ„ ProducerMetadata мҷҖ мң мӮ¬н•ҳкІҢ TopicмқҙлӮҳ Partition м •ліҙлҘј к°Җм§Җкі  мһҲлӢӨ.

л©”мӢңм§Җмқҳ KeyмҷҖ ValueлҘј м§Җм •лҗң SerializerлҘј нҶөн•ҙ м§Ғл ¬нҷ”н•ңлӢӨ.

мң„ мҪ”л“ңмҷҖ к°ҷмқҙ м§Җм •лҗң KeySerializerмҷҖ ValueSerializerлҘј нҶөн•ҙ л©”мӢңм§Җмқҳ лҚ°мқҙн„°лҘј м§Ғл ¬нҷ”н•ңлӢӨ. мқҙ кіјм •м—җм„ң мҳӨлҘҳк°Җ л°ңмғқн•  кІҪмҡ° SerializationExceptionмқҙ л°ңмғқн•ҳкІҢ лҗңлӢӨ.

л©”мӢңм§Җк°Җ л°ңн–үлҗ  Partitionмқ„ кІ°м •н•ңлӢӨ.

partition л©”м„ңл“ңлҘј нҶөн•ҙ л©”мӢңм§ҖлҘј м „мҶЎн•  Partitionмқҳ ID к°’мқ„ м–»м–ҙмҳЁлӢӨ.

мқҙ partition н•ЁмҲҳлҠ” л©”мӢңм§Җмқҳ Keyк°Җ м„Өм •лҗҳм–ҙ мһҲлҠ”м§Җ м•„лӢҢм§Җм—җ л”°лқј мҷ„м „нһҲ лҸҷмһ‘л°©мӢқмқҙ лӢ¬лқјм§ҖкІҢ лҗңлӢӨ.

  • Keyк°Җ м„Өм •лҗҳм–ҙ мһҲлҠ” кІҪмҡ°: Keyмқҳ н•ҙмӢң к°’мқ„ нҶөн•ҙ нҳ„мһ¬ мӮ¬мҡ© к°ҖлҠҘн•ң нҢҢнӢ°м…ҳ мӨ‘ н•ҳлӮҳлҘј м„ нғқн•ңлӢӨ. мҰү, лҸҷмқј KeyлҘј к°Җм§„ л©”мӢңм§ҖлҠ” лҸҷмқјн•ң нҢҢнӢ°м…ҳмңјлЎң м „мҶЎлҗңлӢӨ.
  • Keyк°Җ м„Өм •лҗҳм–ҙ мһҲм§Җ м•ҠмқҖ кІҪмҡ°: StickyPartitionCache к°қмІҙлҘј нҶөн•ҙ нҢҢнӢ°м…ҳмқҙ кІ°м •лҗңлӢӨ. Topic кё°мӨҖмңјлЎң л§ӨлІҲ к°ҷмқҖ нҢҢнӢ°м…ҳ лІҲнҳёлҘј л°ҳнҷҳн•ҳл©°, мІ« нҳём¶ң мӢң Topic лі„лЎң лһңлҚӨн•ң нҢҢнӢ°м…ҳ IDлҘј Cacheн•ҳм—¬ н•ҙлӢ№ к°’мқ„ кі м •мңјлЎң мӮ¬мҡ©н•ңлӢӨ.

Buffer м—ӯн• мқ„ н•ҳлҠ” RecordAccumulator к°қмІҙм—җ л©”мӢңм§ҖлҘј м ҖмһҘн•ңлӢӨ.

RecordAccumulatorлҠ” м—¬лҹ¬ к°ңмқҳ л©”мӢңм§ҖлҘј л°°м№ҳ л°©мӢқмңјлЎң м „мҶЎн•ҳкё° мң„н•ҙ, мӢӨм ң BrokerлЎңмқҳ м „мҶЎ м „м—җ мһ„мӢңлЎң ліҙкҙҖн•ҙл‘җлҠ” BufferмқҙлӢӨ. лӮҙл¶Җм ҒмңјлЎң TopicPartitonмқ„ нӮӨлЎң к°Җм§Җкі  Deque<ProducerBatch>лҘј к°’мңјлЎң к°Җм§ҖлҠ” ConcurrentMap нғҖмһ…мқҳ к°қмІҙ(batches)м—җ л©”мӢңм§ҖлҘј мҢ“м•„лҶ“лҠ”лӢӨ.

м•„лһҳмҷҖ к°ҷмқҙ л©”мӢңм§ҖлҘј batchesм—җ л„Јм–ҙл‘җлҠ” лЎңм§Ғмқ„ мҲҳн–үн•ңлӢӨ. batchesмқҳ нӮӨлҠ” TopicPartition мқҙкё° л•Ңл¬ём—җ, лҸҷмқјн•ң нҶ н”ҪлӘ…кіј нҢҢнӢ°м…ҳ IDлҘј к°Җм§„ л©”мӢңм§Җл“ӨмқҖ н•ҳлӮҳмқҳ ProducerBatchлЎң л¬¶м—¬ м „мҶЎлҗҳкі , Brokerм—җм„ңлҠ” лҸҷмқјн•ң RecordBatchлЎң м ҖмһҘлҗңлӢӨ.

м—¬кё°к№Ңм§Җк°Җ sendлҘј нҳём¶ңн•ҳл©ҙ л°ңмғқн•ҳлҠ” кіјм •мқҙлӢӨ. мӢӨм ң л©”мӢңм§Җмқҳ м „мҶЎмқҖ лі„лҸ„ I/O ThreadлҘј нҶөн•ҙ SenderлқјлҠ” к°қмІҙлҘј нҶөн•ҙ мқҙлЈЁм–ҙм§„лӢӨ.

SenderлҘј нҶөн•ң л©”мӢңм§Җмқҳ м „мҶЎ

SenderлҠ” I/O Threadм—җм„ң мЈјкё°м ҒмңјлЎң RecordAccumulatorмқҳ л©”мӢңм§Җл“Өмқ„ Fetchн•ҳм—¬ Kafka BrokerлЎң м „мҶЎн•ҳлҠ” лӘЁл“ҲмқҙлӢӨ. KafkaProducerлҘј мҙҲкё°нҷ”н•ҳлҠ” кіјм •м—җм„ң I/O Threadк°Җ мғқм„ұлҗҳм–ҙ Senderк°Җ мһҗлҸҷмңјлЎң мӢңмһ‘лҗңлӢӨ.

Sender нҒҙлһҳмҠӨмқҳ мЈјм„қмқ„ ліҙл©ҙ Kafka ClusterлЎңмқҳ л©”мӢңм§Җ м „мҶЎ мҡ”мІӯмқ„ кҙҖлҰ¬н•ҳлҠ” Background Threadлқјкі  м„ӨлӘ…лҗҳм–ҙ мһҲлӢӨ.

мӢӨм ң лЎңм§Ғмқҙ мҲҳн–үлҗҳлҠ” run л©”м„ңл“ңлҘј ліҙл©ҙ м•„лһҳмҷҖ к°ҷмқҙ лЈЁн”„ лӮҙм—җм„ң runOnceлқјлҠ” л©”м„ңл“ңлҘј нҳём¶ңн•ҳлҠ” кІғмқ„ нҷ•мқён•  мҲҳ мһҲлӢӨ.

runOnceлқјлҠ” л©”м„ңл“ң лӮҙм—җм„ңлҸ„ мӢӨм ңлЎң RecordAccumulatorлЎңл¶Җн„° м „мҶЎн•  лҚ°мқҙн„°лҘј Fetchн•ҳм—¬ BrokerлЎң м „мҶЎн•ҳлҠ” м—ӯн• мқ„ мҲҳн–үн•ҳлҠ” sendProducerDataлқјлҠ” л©”м„ңл“ңлҘј нҳём¶ңн•ңлӢӨ.

RecordAccumulatorмқҳ drain л©”м„ңл“ңлҘј нҳём¶ңн•ҳм—¬ м „мҶЎн•  л©”мӢңм§Җ л°°м№ҳл“Өмқ„ Fetchн•ҳкі , sendProduceRequests н•ЁмҲҳлҘј нҳём¶ңн•ҳм—¬ мқҙ л©”мӢңм§Җ л°°м№ҳл“Өмқ„ м „мҶЎн•ҳкІҢ лҗңлӢӨ.

м •лҰ¬

нҳ„м—…м—җм„ң KafkaлҘј мӮ¬мҡ©н•ҳлҠ” кІҪмҡ°лҠ” Spark StreamingмқҙлӮҳ Druid л“ұмқҳ мҳӨн”ҲмҶҢмҠӨлҘј нҶөн•ҙ нҷңмҡ©н•ҳкё° л•Ңл¬ём—җ, Kafka APIлҘј м§Ғм ‘ мӮ¬мҡ©н•  мқјмқҙ м—Ҷм—ҲлӢӨ(л¬јлЎ  мқҙлҹ¬н•ң мҳӨн”қмҶҢмҠӨл“Өл§Ҳм Җ мөңк·јм—җ мӮ¬мҡ©н•ҙлҙӨлӢӨ).

мқјлӢЁ API мһҗмІҙлҘј мӮ¬мҡ©н•ҳл©ҙм„ң м—„мІӯ мү¬мҡҙ мӮ¬мҡ©лІ•(л¬јлЎ  л¶Җк°Җм Ғмқё м„Өм •мқҙлӮҳ Production Levelмқҙ м•„лӢҲкё° л•Ңл¬ём—җ мҳҲмҷё мғҒнҷ©мқҖ мғқкёё мҲҳ мһҲм§Җл§Ң)м—җ н•ңлІҲ лҶҖлһҗкі , лӮҙл¶Җ кө¬мЎ°к°Җ мқҙл ҮкІҢ ліөмһЎн•ҳлӢӨлҠ”лҚ°м—җ л‘җлІҲ лҶҖлһҗлӢӨ.

мұ…м—җ лӮҳмҳЁ лӮҙмҡ©л“Өмқ„ лҚ” мһҗм„ёнһҲ мқҙн•ҙн•  мҲҳ мһҲм–ҙ мўӢм•ҳлҚҳ мӢңк°„мқҙм—ҲлӢӨ.
